# YeongjuTRIG

### 삼각함수의 구조를 시각화하다

YeongjuTRIG는 **영주고등학교 수학 교과 세부능력특기사항(세특) 탐구 활동**을 위해 제작한  
삼각함수 기반 **3D 삼각 측량(Triangulation) 시뮬레이터**입니다.

본 프로젝트는 사인(sin), 코사인(cos), 탄젠트(tan) 함수가  
교과서 속 공식에 그치지 않고,  
**컴퓨터 비전과 자율주행자동차의 거리 추정 기술**에서  
어떤 구조적 역할을 하는지를 이해하기 위해 기획·개발되었습니다.

---

## 📌 프로젝트 기획 배경

수학Ⅱ에서 배우는 삼각함수는  
공식 암기 위주의 학습으로 인해  
실제 활용 맥락을 체감하기 어렵다는 한계가 있습니다.

특히 삼각함수의 탄젠트 성질이  
현실 세계의 거리와 위치 계산에 어떻게 사용되는지에 대한 이해가 부족하다고 느껴  
이를 **시각적·프로그래밍적 탐구**로 확장하고자 본 프로젝트를 진행하였습니다.

---

## 🎯 탐구 목표

- 삼각함수(sin, cos, tan)를 이용한 **삼각 측량의 원리 이해**
- 두 시점(카메라)에서의 각도 정보만으로  
  물체의 위치와 거리를 계산할 수 있음을 수학적으로 확인
- 각도 오차가 거리 오차로 증폭되는 현상 분석
- 자율주행자동차에서 사용되는 거리 추정 기술의 **수학적 기반 탐구**

---

## 🧠 핵심 개념

본 프로젝트는 다음 수학 개념을 중심으로 구성되었습니다.

- 삼각함수: sin, cos, tan
- 삼각 측량(Triangulation)
- 각도와 거리의 관계
- 좌표를 이용한 위치 추정
- 이상적인 조건에서의 수학적 모델링

> ※ 본 시뮬레이터는  
> 카메라 위치와 각도가 정확히 알려져 있고  
> 왜곡이 없는 **이상적인 조건**을 가정한 모델입니다.

---

## 🖥️ 주요 기능

- **3D 공간 시각화**

  - 두 개의 가상 카메라와 물체를 3차원 공간에 배치
  - 전체 공간을 자유롭게 회전·관찰 가능

- **카메라 시점 화면**

  - 좌측 / 우측 카메라 각각의 시점 제공
  - 실제 자율주행 차량 전면 카메라 구조를 단순화한 모델

- **다중 물체 선택**

  - 여러 물체 중 하나를 선택하여 삼각 측량 수행
  - 선택된 물체에 대해 시선 직선, 각도, 거리 표시

- **삼각함수 기반 계산 과정 표시**

  - tan(θ₁) = y / x
  - tan(θ₂) = y / (b − x)
  - 연립 계산을 통해 위치와 거리 추정 과정 단계별 출력

- **오차 민감도 탐구**
  - 각도를 미세하게 변경했을 때 거리 값이 크게 변하는 현상 확인

---

## 🚗 자율주행자동차와의 연관성

자율주행자동차는 두 개 이상의 카메라를 사용해  
물체를 바라보는 각도의 차이를 계산하고,  
이를 기반으로 물체까지의 거리를 추정합니다.

YeongjuTRIG는 이러한 실제 기술을 그대로 구현하는 것이 아니라,  
그 **수학적 원리(삼각 측량)**를  
이상적인 조건에서 단순화하여 탐구할 수 있도록 설계된 모델입니다.

수학적 개념을 프로그래밍과 시각화를 통해 탐구하는 경험을 할 수 있었습니다.
